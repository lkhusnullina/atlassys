### Деплой проекта: https://lkhusnullina.github.io/atlassys/
#### Запуск проекта

#### Скачайте архив проекта или склонируйте репозиторий

```sh
git@github.com:lkhusnullina/atlassys.git
```

#### Установите зависимости проекта

```sh
npm install
```
#### Запустите проект командой

```sh
npm run start
```
